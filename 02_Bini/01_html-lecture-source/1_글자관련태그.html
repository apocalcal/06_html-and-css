문단을 나누는 태그

아래는 p 으로 작성 된 문장임

문단 영역을 나누는 태그로는 p 태그와 pre 태그가 있음
p 태그는 문단 영역을 나누는 태그이지만 한 개의 공백만 표시하며
줄 바꿈 입력을 별도의 태그로 지정해 주며 pre 태그는 여러 칸 띄우기 혹은
줄 바꿈 등을 포함하여 내용 그대로 표현해주는 태그임
function foo() { console.log('foo'); }
