문단을 나누는 태그

문단 영역을 나누는 태그로는 p태그와 pre 태그가 있습니다.
p 태그는 문단영역을 나누는 태그이지만 한개의 공백만 표시하며
줄 바꿈 입력을 별도의 태그로 지정해 주어 pre 태그는 여러 칸 띄우기 혹
줄바꿈 등을 포함하며 내용 그대로 표현하는 태그입니다. function foo() { console.log('hello world!'); }
