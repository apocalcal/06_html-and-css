화면 배치 방법 변경경

블럭요소와 인라인요소의 속성을 변경하여 배치방식을 변경할 수 있다.

블럭 요소를 인라인요소로 변경

>>> display: inline 테스트
인라인 요소로 변경되면 width와 height속성이 무시된다.

블럭요소를 인라인블럭 요소로 변경

>>> display: inline-block 테스트
영역은 인라인요소로 변경되지만 내내용은 블럭요소가 되어 width와 height속성을 설정할 수 있다.(요소간에 작은 간격이 존재)

인라인 요소를 블럭요소로 변경

>>> display: block 테스트
인라인 요소를 블럭 요소로 변경하면 width와 height값을 지정할 수 있고 세로로 쌓인다.
